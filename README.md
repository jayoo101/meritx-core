# ⚙️ MeritX Core

> **"Don't Trust, Verify."** This repository contains the immutable core smart contracts for the **MeritX Protocol**, deployed on the Base Network. 

MeritX is a decentralized, transparent, and fair-launch infrastructure designed to eliminate centralized gatekeepers. We believe that code is law, and transparency is the ultimate foundation of trust.

## 🏗️ Architecture

- **Price-of-Proof (PoP) Engine**: Allocation is not granted by fiat; it is earned through historical on-chain contribution (Gas Burned).
- **Sybil Resistance**: Hardcoded architectural defenses against botting and whale monopolization.
- **Fair Launch Logic**: Dynamic Pro-Rata distribution ensuring absolute mathematical fairness.

## 📂 Repository Structure

- `blockchain/`: Contains the Solidity source code, interfaces, and deployment scripts for the MeritX IAO logic.
- *Note: The frontend client and API gateways remain closed-source to protect the ecosystem's user experience.*
